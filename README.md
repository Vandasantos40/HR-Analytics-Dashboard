# 📊 HR Analytics Dashboard

## Overview

This interactive HR Analytics Dashboard was developed using **Power BI** to analyze workforce data and provide actionable insights into employee attrition, demographics, and organizational performance.

The dashboard was designed to support data-driven decision-making by transforming raw HR data into clear and interactive visualizations.

---

## Business Objectives

* Monitor employee attrition.
* Analyze workforce demographics.
* Identify departments with the highest turnover.
* Evaluate overtime impact on attrition.
* Support strategic HR decision-making.

---

## Key Performance Indicators (KPIs)

* Total Employees
* Attrition Employees
* Attrition Rate
* Average Age
* Average Monthly Income

---

## Dashboard Features

* Employee Attrition by Department
* Attrition by Job Role
* Attrition by Gender
* Attrition by Marital Status
* Overtime Analysis
* Interactive Filters
* Executive Key Insights

---

## Tools & Technologies

* Power BI
* DAX
* Power Query
* Data Modeling
